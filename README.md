# Oscar Award-Winning Films Web Crawler

## Project Overview

This project is a web crawler application that scrapes data about Academy Award–winning films from Wikipedia and displays the extracted information through a Flask web application.

The crawler collects structured film data using Python Regular Expressions (`re`) and stores the results in JSON format.

The Academy Awards are presented annually by the Academy of Motion Picture Arts and Sciences (AMPAS).

---

## Data Source

Wikipedia:
https://en.wikipedia.org/wiki/List_of_Academy_Award–winning_films

Each film page is individually crawled to extract detailed information.

```
    ├── script.js                 ← search, filter, sort, pagination

    └── oscar.png
```

## Regular Expressions (10 patterns — all in extractor.py)

| # | Name | What it extracts |
|---|------|-----------------|
| RE-1 | `extract_director` | Director name(s) following "Directed by" label |
| RE-2 | `extract_starring` | Full cast list from Starring section |
| RE-3 | `extract_production_companies` | Production company names |
| RE-4 | `extract_distributor` | Distribution company names |
| RE-5 | `extract_running_time` | Film duration in minutes |
| RE-6 | `extract_country` | Country or countries of origin |
| RE-7 | `extract_budget` | Production budget with currency symbols (₩, $, €, £) |
| RE-8 | `extract_box_office` | Box office gross with currency symbols |
| RE-9 | `extract_language` | Film language(s) |
| RE-10 | `reference_cleanup` | Strips Wikipedia footnote markers [1], [a], [b] |

## How to Run the Project

```bash
# Clone the repo
git clone https://github.com/ayemmk21/Web_crawler
cd OSCAR_MOVIES

# Create and activate virtual environment
python -m venv .venv
source .venv/bin/activate        # Mac/Linux
.venv\Scripts\activate           # Windows

# Install dependencies
pip install -r requirements.txt

# Step 1: Crawl Wikipedia (~2-3 minutes, scrapes 149 films)
python scraper.py

# Step 2 (Optional): Fix missing/malformed data
python fix_budgets.py
python fix_production_companies.py

# Step 3: Start the web app
python app.py
# Open: http://127.0.0.1:5000
```

## Extracted Data Fields

| Field | Description |
|-------|-------------|
| `title` | Film title |
| `year` | Oscar ceremony year |
| `awards` | Number of awards won |
| `nominations` | Number of nominations |
| `director` | Director(s) |
| `starring` | Main cast |
| `production_companies` | Production companies |
| `distributor` | Distribution companies |
| `running_time` | Film duration |
| `country` | Country of origin |
| `budget` | Production budget |
| `box_office` | Box office gross |
| `language` | Film language(s) |
| `poster` | Poster image URL |

These fields are extracted using multiple regular expression patterns applied to the film’s infobox HTML.

---

## Features

- **Oscar films** scraped from Wikipedia (1927–2024)
- Film **card grid** displaying posters, title, year, and win count
- **Search** by title, director, or cast
- **Filter** by year (1927–2024)
- **Sort** A-Z / Z-A / Default
- **Pagination** (24 films per page)
- Individual **film detail pages** with full extracted info
- REST API at `/api/films`
